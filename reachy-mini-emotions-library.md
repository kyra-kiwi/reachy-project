## Reference

- **Dataset:** [pollen-robotics/reachy-mini-emotions-library](https://huggingface.co/datasets/pollen-robotics/reachy-mini-emotions-library) on Hugging Face.
- **File tree (source of move names):** [Dataset tree – main branch](https://huggingface.co/datasets/pollen-robotics/reachy-mini-emotions-library/tree/main) — move names are the stems of the `.json` files in the repo.
- **In code:** Move names are what `RecordedMoves.list_moves()` returns when loading this dataset (see `reachy_mini.motion.recorded_move.RecordedMoves` in the Reachy Mini Python SDK).
